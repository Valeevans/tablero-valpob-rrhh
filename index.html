Instrucciones para crear el Google Sheets de VALPOB

Creá un nuevo Google Sheets con las siguientes hojas. Cada hoja se conecta a una pestaña del tablero.

  1. Abrí Google Sheets y creá un archivo nuevo llamado RRHH_VALPOB_2026
  2. Creá estas 4 hojas (solapas): Formacion Perfiles Desempenio KPIs
  3. En la hoja Formacion usá estas columnas: ID | Tema | Categoria | DirigidoA | Meses (ENE…DIC, un estado por celda) | Responsable | Obs
  4. En la hoja Perfiles: Puesto | Area | Tipo | Formacion | ExpMin | Competencias (una por columna con nivel 0-4)
  5. En la hoja Desempenio: Puesto | Area | Puntaje | Competencias | Estado | Observaciones
  6. En la hoja KPIs: Codigo | Indicador | Meta | ValorActual | Estado | Periodo
  7. Abrí Extensiones → Apps Script, pegá el código del Web App y publicalo como aplicación web con acceso para cualquier persona.
  8. Copiá la URL generada y pegala en el campo "URL del Apps Script" arriba.

💡 El tablero carga los datos en tiempo real al hacer clic en "Sincronizar". Si no hay conexión, muestra los datos precargados del plan 2026.

Código Apps Script para pegar en Google Sheets
function doGet(e) {
  const ss = SpreadsheetApp.getActiveSpreadsheet();
  const sheets = ['Formacion','Perfiles','Desempenio','KPIs'];
  const out = {};
  sheets.forEach(name => {
    const sh = ss.getSheetByName(name);
    if (!sh) return;
    const [headers, ...rows] = sh.getDataRange().getValues();
    out[name] = rows.map(r => {
      const obj = {};
      headers.forEach((h, i) => obj[h] = r[i]);
      return obj;
    });
  });
  return ContentService
    .createTextOutput(JSON.stringify(out))
    .setMimeType(ContentService.MimeType.JSON);
}
